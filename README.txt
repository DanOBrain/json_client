Веб-Клиент
Автор: Давлетов Динис
Использование: main.go [параметры]

Параметры:
  -user int         Фильтр по ID пользователя
  -limit int        Лимит результатов (по умолчанию 3)

Примеры:
  json-client -user=1 -limit=5
  json-client -user=2 -limit=3

API: https://jsonplaceholder.typicode.com
